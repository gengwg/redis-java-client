# redis-java-client

A simple Java client for Redis cluster.

## Usage

```bash
$ mvn clean package
$ java -cp target/redis-client-1.0-SNAPSHOT.jar com.mycompany.app.RedisClient
```

## Examples

```
$ mvn clean package
$ java -cp target/redis-client-1.0-SNAPSHOT.jar  com.mycompany.app.RedisClient
Connection to server sucessfully
The number of nodes in cluster is: 6
foo: bar
hello: world
```
